**Most useful shortcuts**

| Shortcut             | Use                               |
|----------------------|----------------------------------|
| Ctrl + Shift + P    | Search all IDE functions/commands |
| Ctrl + Shift + F    | Find in all files                |
| Ctrl + `            | Open terminal                    |
| Shift + Alt + Up/Down | Duplicate line above/below     |
| Ctrl + /            | Comment out line                 |
| Ctrl + H            | Find & Replace                   |
